Tipos de variables  primitivas:

variable: espacio de memoria donde se almacena informacion.
byte: es para numeros de -128 hasta 127 tamaño 8
short es para numeros -32.768 a 32.767 tamaño 16
int : se puede utilizar para grandes cantidades que seria un tamaño de 32 numeros
long : para numeros, tamaño 64
float : es para decimal pero alterminar el numero se debe poner una letra f ejemplo (3.45f)
double: decimales con un tamaño mas amplio que el float.
char : es para guardar caracter , que es caracter todo aquello que encuentres en el teclado cualquier tecla.
boolean: es la que sirve para saber si es verdad o falso.
